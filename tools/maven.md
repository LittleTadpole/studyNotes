#### Maven

一款服务于java平台的自动化构建工具



#### 配置与命令

 配置环境变量 (操作略);

查看命令:  mvn -v

==与构建过程相关的maven命令,必须进入pom.xml所在的目录==

- 清理 mvn clean
- 编译 mvn compile
- 编译测试程序 mvn test-compile
- 执行测试: mvn test
- 打包:  mvn package
- 安装 mvn install , 在待安装工程目录下,执行此命令
- 生成站点: mvn site

#### POM

Project Object Model 项目对象模型, maven工程的核心配置文件.

直接右键可以出来maven命令快捷方式
